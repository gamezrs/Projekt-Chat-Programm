\hypertarget{index_intro_sec}{}\doxysection{\texorpdfstring{Einführung}{Einführung}}\label{index_intro_sec}
Simple\+Chat ist ein Peer-\/to-\/\+Peer-\/\+Chat-\/\+Programm zur lokalen Kommunikation in IP-\/\+Netzwerken. Es basiert auf dem {\bfseries{Simple Local Chat Protocol (SLCP)}} und ermöglicht die Übertragung von\+:
\begin{DoxyItemize}
\item Textnachrichten
\item Bildern
\item Systemnachrichten
\end{DoxyItemize}

\hypertarget{index_features_sec}{}\doxysection{\texorpdfstring{Hauptfunktionen}{Hauptfunktionen}}\label{index_features_sec}

\begin{DoxyItemize}
\item {\bfseries{Automatische}} Peer-\/\+Entdeckung via UDP-\/\+Broadcast
\item {\bfseries{Zuverlässiger}} Bildtransfer mit Handshake-\/\+Mechanismus
\item {\bfseries{CLI-\/\+Oberfläche}} für einfache Bedienung
\item {\bfseries{Modulare}} Architektur (siehe \doxysectlink{index_architecture_sec}{Systemarchitektur}{1})
\end{DoxyItemize}\hypertarget{index_architecture_sec}{}\doxysection{\texorpdfstring{Systemarchitektur}{Systemarchitektur}}\label{index_architecture_sec}
